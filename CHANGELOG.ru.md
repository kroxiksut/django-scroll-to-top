# История изменений

Все заметные изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект следует [семантическому версионированию](https://semver.org/lang/ru/).
Пока версия остаётся в серии `0.x`, публичный API может меняться между
минорными релизами.

## [0.1.0] - 2026-06-27

Первый публичный релиз (бета) **django-scroll-to-top** — переиспользуемого
Django-приложения, которое отображает один доступный настраиваемый элемент
«наверх» на страницах сайта и в стандартной Django Admin.

Полный набор возможностей и настройку см. в [README](README.ru.md) и
[документации](docs/README.ru.md). Начиная с этого релиза файл фиксирует
изменения между версиями, а не пересказывает, что делает пакет.
