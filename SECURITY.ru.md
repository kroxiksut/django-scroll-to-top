# Политика безопасности

## Поддерживаемые версии

Проект находится в серии `0.x` (бета). Исправления безопасности выпускаются для
последнего опубликованного релиза `0.x`. Чтобы получать исправления,
обновляйтесь до последней версии.

| Версия | Поддерживается |
| --- | --- |
| 0.1.x | ✅ |
| < 0.1 | ❌ |

## Сообщение об уязвимости

Пожалуйста, **не** сообщайте о проблемах безопасности в публичном issue.

Сообщайте приватно через механизм GitHub **«Report a vulnerability»** на вкладке
**Security** репозитория:
<https://github.com/kroxiksut/django-scroll-to-top/security/advisories/new>.
Так создаётся приватный security advisory, видимый только сопровождающим. Если
этот механизм недоступен, напишите на <fmalkov91@gmail.com>.

Пожалуйста, укажите затронутые версии, шаги воспроизведения и наблюдаемое
воздействие. Мы стремимся подтвердить получение в течение нескольких дней и
согласовать с вами исправление и сроки раскрытия.

Ключевые зоны риска: sanitization SVG, авторизация в админке, шаблонный
рендеринг, хранение dismissal state, endpoint same-origin stylesheet и runtime
DOM hooks.
