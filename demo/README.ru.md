# Демо-сайт

[English version](README.md)

В этом каталоге находится самостоятельный Django-проект для локальной
демонстрации.

## Запуск

```console
python -m pip install .
python demo/manage.py migrate
python demo/manage.py runserver
```

Перед запуском demo зависимости должны быть установлены в активное Python-
окружение.

Первый `migrate` подготавливает данные:

- рабочего суперпользователя `admin` / `admin`;
- опубликованную конфигурацию scroll-to-top по умолчанию для публичного сайта;
- опубликованную конфигурацию scroll-to-top по умолчанию для Django admin.

## Содержимое demo

- Публичные страницы только на английском.
- Django admin использует встроенную английскую и русскую локализацию Django.
- При первом `migrate` demo-база создаёт суперпользователя с учётными данными
  `admin` / `admin`.
- Содержимое demo оригинальное и предназначено для локальной проверки прокрутки
  и вёрстки.
- Страница `Obstacles` (`/obstacles/`) демонстрирует движок коллизий и
  необязательный адаптер препятствий: cookie-баннер в правом нижнем углу с
  кнопками открытия, сворачивания и закрытия, виджет чата, toast и
  закреплённую мобильную панель навигации как одновременные препятствия.
  Кнопка `Toggle collision debug` визуализирует прямоугольники препятствий.

## Переключатель языка в админке

В demo-админке есть компактный переключатель английский/русский в области
branding.

- Он использует стандартный endpoint Django `set_language`, поэтому не нужна
  отдельная логика авторизации или дополнительный URL.
- Переключатель оформлен отдельной demo-таблицей стилей, а не стандартными
  кнопками админки.
- Активный язык подсвечивается, и рядом показывается короткая подсказка о
  locale, чтобы текущее состояние читалось сразу.
- Это относится только к demo-проекту; сам пакет по-прежнему опирается на
  обычное поведение локализации Django.

## Значения scroll-to-top по умолчанию

Demo подготавливает два опубликованных профиля конфигурации пакета:

- и публичный профиль `site`, и профиль `admin` стартуют с одинаковой зелёной
  кнопкой, залитой стрелкой и контрастными цветами, поэтому кнопка в админке
  совпадает с тем, что по умолчанию видит публичный сайт;
- сохранение опубликованной ревизии `admin` сразу копирует визуальные поля в
  публичный профиль `site`, поэтому backend становится живым предпросмотром
  того же самого контролла;
- demo хранит два профиля отдельными записями, чтобы контракт пакета оставался
  в силе, но сама связка делает их синхронизированной конфигурацией.

Это делает demo реалистичным и не даёт публичному сайту и админке расходиться
во время сценария.
