# Документы для ИИ: django-scroll-to-top

Этот раздел содержит инструкции для ИИ-агентов по интеграции
`django-scroll-to-top`.

Сами инструкции поддерживаются **только на английском языке**, чтобы промпты,
ключевые слова и контракты интеграции оставались единообразными для ИИ-инструментов:

- [README.md](./README.md) — что это и как подключить к современным ИИ.
- [integration.md](./integration.md) — установка, подключение, URL, тег.
- [presentation.md](./presentation.md) — безопасное изменение внешнего вида (конфиг в админке).
- [runtime.md](./runtime.md) — браузерный слой, `window.djstt`, DOM-события, SPA.
- [icons.md](./icons.md) — встроенные/разработческие/загруженные иконки, безопасность SVG, атрибуция.

Как подключить к ИИ (Claude Code, Cursor, ChatGPT и т. п.) — см. раздел
«How to connect these to a modern AI» в [README.md](./README.md).

Примечание по quality-gate: основной инструмент статической типизации — `pyright`.
