# Процесс перевода

Английский является каноническим языком для комментариев в коде, docstring,
документации и Django-строк `msgid`. Русский является первым поддерживаемым
переводом.

Когда меняются интерфейсные строки:

1. Обновите английскую исходную строку.
2. Обновите русский каталог перевода.
3. Пересоберите скомпилированные `.mo` в том же change set.
4. Синхронизируйте связанную английскую и русскую документацию.
