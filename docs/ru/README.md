# Документация: django-scroll-to-top

[English version](../en/README.md)

`django-scroll-to-top` — переиспользуемое Django-приложение, которое отрисовывает
один доступный и настраиваемый элемент «прокрутка наверх» на страницах сайта и в
стандартной админке Django. Внешний вид и поведение хранятся в конфигурации на
основе базы данных и редактируются через админку Django; настройки Django
отвечают только за установку и инфраструктуру.

![Элемент «прокрутка наверх» по умолчанию на демо-сайте](../assets/shared/default-button-demo-site.png)

> **Статус: до релиза.** Код пакета, модель данных, админка, браузерный слой и
> набор тестов уже реализованы в этом репозитории, но дистрибутив ещё не
> опубликован на PyPI. Команды установки описывают планируемый стабильный
> контракт и могут измениться до первого публичного релиза.

## Инструкции для ИИ-агентов

Англоязычные инструкции для ИИ-агентов по подключению пакета в проект лежат в
[docs/ai/](../ai/README.ru.md). Сами инструкции — только на английском, чтобы
промпты и контракты были единообразны для ИИ-инструментов.

**Как подключить к современным ИИ:** это обычный Markdown. Укажите
терминальному агенту (Claude Code, Codex) на `docs/ai/`, сошлитесь на них из
своего `AGENTS.md` / `CLAUDE.md` / `.cursor/rules` или вставьте отдельный гайд
в чат-ассистент (Claude.ai, ChatGPT, Gemini) как контекст. Пошагово по
инструментам — раздел
[«How to connect»](../ai/README.md#how-to-connect-these-to-a-modern-ai).
